# goofy ahh slack bot

built for the hackclub thingy

## commands

`/goofyahhbot-ping` - pings the bot and checks the latency

`/goofyahhbot-help` - prints this help message, which is located in a text file

`/goofyahhbot-ismysiteup` - checks if https://bestspark.org is up

`/goofyahhbot-no` - returns a response from no as a service

`/goofyahhbot-paramtest` - returns the parameters you give it to you.

`/goofyahhbot-yes` - returns a response from yes as a service. add nothing, `funny`, `corporate`, or `sarcastic` to get different responses.

(yes this is literally just the help commands so... yeah)
